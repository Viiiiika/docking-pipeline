Docking pipeline (smina) — ноутбук

Репозиторий содержит воспроизводимый ноутбук для молекулярного докинга малых молекул к белковым мишеням.

Пайплайн включает:
- загрузку/подготовку структуры белка-мишени (PDB),
- подготовку лигандов (из SMILES → 3D → PDBQT),
- подготовку рецептора (PDB → PDBQT),
- запуск докинга через **smina**,
- сбор результатов (лучший `affinity`, позы) и сохранение файлов для визуализации.
